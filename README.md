# Regression Algorithms Project

This project applies two regression algorithms to a dataset using Python.

## Structure

- `Main.py`: Main script for loading data, training models, and evaluating results.
- `Data/`: Dataset files.
- `Assets/`: Generated plots, images, or other project assets.
- `Base/`: Local base files ignored by Git.

## Usage

Run the main script with:

```bash
python Main.py
```
